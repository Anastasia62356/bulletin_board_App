# bulletin_board_App


## 概要
ASP.NET を用いて作成した掲示板アプリです。  
ユーザー登録、ログイン、投稿機能などを実装し、掲示板として基本的な操作ができるようにしました。

## 主な機能
- ユーザー登録
- ログイン / ログアウト
- 投稿の作成
- 投稿内容の表示
- 掲示板形式での閲覧
- 簡易的な返信

## 使用技術
- C#
- ASP.NET Web Forms
- Visual Studio
- HTML / CSS
- SQL Server

## 工夫した点
- ユーザー登録機能とログイン機能を実装し、利用者ごとの利用を想定した構成にしました。
- 掲示板として必要な投稿・表示の流れが分かりやすくなるように画面を構成しました。
- ASP.NET Web Forms の画面遷移やフォーム処理の流れを意識して実装しました。

## 今後の改善点
- 投稿の編集・削除機能の追加
- UIデザインの改善
- エラーメッセージ表示の改善
- セキュリティ面の強化

## 制作意図
Web アプリケーション開発の基礎とデータベースを学ぶことを目的として制作しました。  
ユーザー管理と投稿機能を備えた掲示板アプリを通して、フォーム処理やデータ管理の流れを学びました。
